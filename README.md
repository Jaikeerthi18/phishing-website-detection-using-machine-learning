# Phishing Website Detection System using Machine Learning

## Project Overview

This project is developed to detect whether a website URL is phishing or legitimate using Machine Learning techniques.  
The system extracts URL-based features and predicts the website status using a trained ML model.

---

# Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Gradio

---

# Features

- Detects phishing websites
- Machine Learning based prediction
- User-friendly interface
- Real-time URL analysis
